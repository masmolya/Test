Данный телеграм бот написан для проведения тест-опросов для n-ого количества пользователей
 и последующей отправкой их ответов в закрытую группу(созданную для удобства, чтобы не засорять личные сообщения).
Так же, администратор будет получать id опрошенных и
иметь возможность сделать данный тест-опрос приватным (для исключения возможности 
списывания посредством "фейкового аккаунта"/изменения "уникального имени" или иных манипуляций).
